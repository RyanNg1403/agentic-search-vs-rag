
Session summary service in packages/core/src/services/sessionSummaryService.ts generates AI summaries of chat sessions using BaseLlmClient. SessionSummaryService.generateSummary() creates one-sentence summaries (max 80 chars) focused on user intent. You must use sliding window selection: takes first N/2 and last N/2 messages (DEFAULT_MAX_MESSAGES=20). Filters to user/gemini messages only, truncates long messages to MAX_MESSAGE_LENGTH (500 chars). Uses SUMMARY_PROMPT template. Timeout protection via DEFAULT_TIMEOUT_MS (5000ms). Returns null on failure. Ideal for session restoration and session list display in --list-sessions.
